Sunbreak Privacy Policy

Secure BYOAI — Chrome Extension

Effective date: March 3, 2026

The short version: Sunbreak works entirely inside your browser. Your prompts and personal data are never stored, transmitted, or shared with anyone — including us. The extension makes zero network requests.

1. What Sunbreak Does

Sunbreak detects sensitive personal information — such as email addresses, phone numbers, credit card numbers, Social Security numbers, and API keys — inside prompts you write on supported AI tool pages (ChatGPT, Claude, and Gemini). When a potential exposure is found, the extension warns you and offers one-click redaction before the prompt is sent.

All classification is performed locally in your browser using pattern matching. No data ever leaves the browser.

2. Data We Read

To perform its function, Sunbreak reads the text you type into prompt input fields on the following pages:

  • chatgpt.com and chat.openai.com (ChatGPT)
  • claude.ai (Claude)
  • gemini.google.com (Gemini)

This text is scanned locally and in real time. It is never stored and never transmitted anywhere. Once the scan completes, the raw prompt text is immediately discarded from memory.

3. Data We Store

Sunbreak stores a small amount of data in chrome.storage.local (your browser's local extension storage). This data never leaves your device.

Stored Item Example Purpose
Aggregate detection statistics "5 emails detected today" Powers the personal dashboard
User settings and preferences Enabled detectors, custom keywords Remembers your configuration
Activity log entries "Credit card pattern blocked on ChatGPT" Lets you review past warnings

We never store raw prompt text, personal data values, or any content you type. Only aggregate counts and metadata (such as detection category and timestamp) are retained.

4. Data We Share

None. Sunbreak makes zero network requests. There are no servers, no APIs, no analytics endpoints, and no telemetry. Your data stays on your machine and nowhere else.

5. Third-Party Services

Sunbreak does not integrate with, send data to, or receive data from any third-party service. There are no analytics tools, no crash reporters, no advertising SDKs, and no external dependencies at runtime.

6. How to Delete Your Data

You can remove all data Sunbreak has stored at any time:

  • Uninstall the extension — Chrome automatically deletes all associated chrome.storage.local data when an extension is removed.
  • Clear extension storage manually — Go to chrome://extensions, find Sunbreak, click Details → Clear storage.
  • Use the in-app setting — Open the Sunbreak dashboard and use the "Clear all data" option in Settings.

7. Permissions Explained

Sunbreak requests the minimum set of Chrome permissions needed to function:

  • Active Tab / Host permissions — To read prompt text on supported AI tool pages only.
  • Storage — To save your settings and aggregate statistics locally.

The extension does not request permissions for browsing history, bookmarks, downloads, or any data beyond what is listed above.

8. Chrome Web Store Compliance

This extension complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. Specifically:

  • User data is used solely to provide the extension's core functionality (detecting sensitive data in prompts).
  • User data is not transferred to third parties.
  • User data is not used for advertising or creditworthiness purposes.
  • User data is not sold to third parties and is not used by any entity other than the extension's core, user-facing features.

9. Children's Privacy

Sunbreak is not directed at children under 13 and does not knowingly collect personal information from children.

10. Changes to This Policy

If this privacy policy is updated, the revised version will be published at this URL with an updated effective date. Because Sunbreak collects no contact information, we cannot notify users directly — please check this page periodically.
