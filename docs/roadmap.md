# Sunbreak Roadmap (v1) — Archived

> **This roadmap is superseded by [Roadmap v2](roadmap-v2/README.md).**
>
> The v2 roadmap reorganizes the project around 8 product-driven phases
> with an explicit execution plan. All planning and prioritization should
> reference v2.

## What was completed under v1 numbering

- Phases 1–5: scaffolding, DOM observation, classifier, overlay UI, dashboard
- Post-phase refinements: inline detection, smart tokens, reversible masking, UI redesign, extended widget, design system refactor
- Phase 6a (capability flags) and 6b (live E2E testing)
- Epic 3 (masking productization), Epic 5 (recovery & provider guidance)

## Known issues carried forward

- **E4:** Widget visible when extension disabled
- **T3.3:** Site "Copy" button bypasses clipboard interceptor
- **Untested:** E5 (tab suspension/wake), A2 (screen reader), A3 (reduced motion)
